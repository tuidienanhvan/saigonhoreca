# Status Row Template / Mẫu Dòng Status

```markdown
| T-YYYYMMDD-001 | Gemini | active | Short scope | active/T-YYYYMMDD-001-task.md | not-run | Notes |
```

Columns / Cột:

- ID
- Owner / agent phụ trách
- State / trạng thái
- Scope / phạm vi
- Dossier / file task
- Gates / trạng thái kiểm tra
- Notes / ghi chú
